CSS এর filter property use করে সহজেই image এর color change করা যায়।

img{
filter: grayscale(100%);
}

Multiple filter একসাথে use করা

img {
  filter: grayscale(100%) blur(2px);
}

img {
  filter: contrast(150%);
}

Sepia effect (পুরোনো ছবির মতো)

img {
  filter: sepia(100%);
}

img {
  filter: brightness(70%);
}
