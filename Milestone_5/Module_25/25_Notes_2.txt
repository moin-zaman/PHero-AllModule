1. document.querySelectorAll('.fruitsContainer')
--> এটা return করবে একটা NodeList [section.fruitsContainer]।
কোনো element না পেলে return করবে একটা empty NodeList []।

2. document.querySelector('.fruitsContainer')
--> এটা return করবে fruitsContainer class যুক্ত html tag।

3. document.querySelector('.fruitsContainer li')
--> এটা return করবে .fruitsContainer class এর ভেতরের সবগুলো li এর মধ্যে প্রথম li tag।

4. document.querySelector('')
--> এখানে কিছু না পেলে return করবে null।
