Box Model : web page এর ভেতরে একটা element width, height, margin, padding, border দিয়ে আসলে কতটুকু জায়গা দখল করে আছে সেটাই হচ্ছে Box Model.

** Box Model block level elements এ কাজ করে কিন্তু inline elements এর ক্ষেত্রে পুরোপুরি support করে না। ***

** Inline elements গুলো তার parent এর padding, width, height গুলোকে overflow করে মাঝে মাঝে বাইরে বের হয়ে যাওয়ার উপক্রম হয়ে যায়। **

Height  X Width = Content Size.

Image, Button ইত্যাদি width, height নিয়ে কাজ করতে পারে কিন্তু বেশিরভাগ inline elements width, height support করে না।


** Inline element শুধু inline element কেই contain করতে পারে কিন্তু block level element block, inline দুই ধরনের elements কেই contain করতে পারে। **

Flex Box: কোনো একটা parent এর child elements কে flexibly control করতে চাইলে flex display property টা use করবো।

** Parent element এ flex use করা মানে হলো যে, parent এর child কে control করা flex parent কে কোনো effect করবে না। **

** After using flex property then parent is called : Flex container **

** child elements are called flex items **

** Inline element-এর parent element এ padding দিলে এর child elements গুলো সেই padding-কে overflow করে মাঝে মাঝে বাইরে বের হয়ে আসতে চায়। **

Pixso-তে width আছে 1920px এবং তোমার window-তে width আছে 1280px।
তাহলে 1920px-এর জন্য যেসব value আছে, সেগুলোকে 1280px-এর জন্য convert করতে হবে।

for width 1280px :

 ==== 1280 * 100 / 1920 == 66.67%

 Height =- 919px for width 1920px;

 for width 1280px, the height will be:

 919 * 66.67 / 100

 or 

 ( 66.67 / 100 = 0.67 (almost))

 919 * 0.67 = 615px


 ** formula : New Size = Original Size × Base Width / Target Width **
